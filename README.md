## Tendermint with simplest abci

There are three ways to implemnet ABCI
* In-process
* ABCI-socket
* GRPC

t-tendermint/abci.py implements the **ABCI-socket** method by listening to \[port\]\(26658 by default\) by running
```
python SimpleABCI.py [port]
```
